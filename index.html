Projects

Botniversaire

(2019)

Botniversaire is a Discord bot that wish anniversaries of the users that entered their birthday. This bot as some command that the users can use like "add", "delete", "setDefault" and others.

This project is made with NodeJS.
I create this project to learn more about NodeJS and also for fun because I think we learn more when having fun than do something just because someone told us to do it. This project is totally Open Source and under Beer-Ware license.


DeltaFrench Clan

(2018)

The DeltaFrench Clan is a seasoned multigaming community. It is through the Francophonie and the online game that they strive to implement their brand for more than 15 years now.

This website is made with HTML, CSS and JS only.
The DeltaFrench Clan migrated to Discord, so their old forum was a bit useless. But, the clan still needed a website to show their brand and inform where to contact them. This website integrate a Discord widget that show who is online and who is on the vocal channels. It also integrate a visualisation of the "News" channel on the Discord server.


Orinox Cloud Web Services

(2016)

Orinox Cloud Web Services is an innovative solution to create & visualize your Engineering & Design projects data. It’s a fully integrated Engineering and Design solution for Engineering projects.

This project is made with PHP7, Symfony3 and Doctrine.
This website link all the Orinox Services connected to their Cloud. On this website you can upload your project and convert it for OMR (Orinox Model Review) to visualize it on phone, tablet or PC, or Orinox Simulator for Serious Gaming on it.


Orinox Website

(2015)

ORINOX is a worldwide software consulting, Plant Design & Engineering firm specializing in digitalization using the AVEVA suite products. They operate across many sectors like oil & gas, nuclear, energy and more...

This project is made with the CMS Joomla.
This website show all the services and products that Orinox provide from the CAO to training or Cloud Engineering. It also show the news about Orinox, quizzes for the news employees and an "intranet" for all the intern documents.
